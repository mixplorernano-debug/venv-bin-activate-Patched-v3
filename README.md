Install

First, create a new directory and set up a virtual environment:
```bash
python -m venv venv
source venv/bin/activate
```
```bash
apt update && upgrade -y 
pkg install git python mpv figlet -y
pip install lolcat
git clone https://github.com/fikrado/qurxin
cd qurxin
chmod +x *
sh install.sh
exit

```
